Tools


Facify


Maya Python tool to automate curve-based eye/mouth rig setup on given sets of vertices.

Features include:

  • Joints aim constrained to locators bound to a high-density curve
  • Wire deformer to drive high-density linear curve with a low-density cubic curve
  • Controls angled to follow joint direction/face curvature
  • Blendshape-based Smart Close system for adjustable blink/close attributes
    • GitHub
    Vimeo

    Bendify


    Maya Python tool to automate bendy joint chain setup on two given joints.

    Features include:

  • Automatic setup of any given number of sub-joints (up to 20)
  • Quadratic weight distribution across parent constraints, for bendy behavior
  • Automatic setup of curveInfo and multiplyDivide nodes, for stretchiness
    • GitHub

    Splinify


    Maya Python tool to automate spline IK setup on two given joints.

    Features include:

  • Automatic setup of any given number of sub-joints (up to 100)
  • Interpolated coloring of control curves based on user-input colors
  • Custom attribute to adjust length of joint chain along spline
    • GitHub

    Kinematify


    Maya Python tool to automate IKFK blend setup on a given joint chain.

    Features include:

  • Automatic creation of IK and FK joint chains
  • Creation of FK control curves with correctly oriented group nodes
  • Setup of "IK" attribute to control IK/FK blending from 0 to 1
