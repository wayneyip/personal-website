Superflick

JavaScript (jQuery, three.js, turn.js, Tween.js), HTML, CSS, Marvel API

Role: Front-End Developer


A Marvel comic book virtual experience I developed in a 2-person team in 48 hours for HackTech 2018.

Superflick allows motion-driven page-flipping through the combined integration of turn.js and Myo Armband technology, and contains an animated 3D environment developed in three.js and Tween.js for an immersive reading experience.

Awarded Disney's Best Hack Using Marvel API for HackTech 2018.

Travelyze

JavaScript, HTML, CSS, Google Maps API, Amadeus API

Role: UI Artist

RetroMe

C++, Qt, Qt Creator

Role: Solo Developer (full-stack programming, UI design, art assets)


A pixelated avatar creator I developed from scratch as my personal project for Summer 2017.

RetroMe uses a customized heapsort algorithm to render character parts in layers, accounting for multi-layered sprites like Hair which renders before and after the Head sprite. This avoids the distinction of "front hair" and "back hair" categories found in most 2D avatar creators. It also contains a color parser -- add any color of your choice by simply popping the RGB values into a text file.

Check out the animated GIF demos and source code of all its functionalities on GitHub!
