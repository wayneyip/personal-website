RetroMe

C++, Qt, Qt Creator · 2D Pixelated Avatar Creator · Personal Project

Role: Solo Developer (full-stack programming, UI design, art assets)


RetroMe is a pixelated avatar creator programmed from scratch in object-oriented C++ and Qt.

Notable features:

  • Uses a customized heapsort to render character parts in layers, accounting for multi-layered sprites like Hair which renders before and after the Head sprite.
  • Uses inheritance and polymorphism to create a multi-layered Hair subclass, which dynamically switches between two priorities during heapsort; this avoids the distinction of "front hair" and "back hair" categories found in most 2D avatar creators.
  • Contains a parser for color palettes -- add any color of your choice by simply popping the RGB values into a text file.


Check out the animated GIF demos and source code of all its functionalities on GitHub!
